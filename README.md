# x20chat - Python P2P Texting Application

A high-performance peer-to-peer texting application built in Python, enabling decentralized communication without relying on central servers.

## Overview

x20chat is a decentralized messaging platform that allows users to send text messages directly to each other over a peer-to-peer network. By eliminating the need for a central server, x20chat provides privacy-focused, secure, and real-time communication capabilities.

## Goals

- Build a fully decentralized P2P messaging system
- Implement secure, encrypted communication between peers
- Create an intuitive user interface for seamless messaging
- Ensure cross-platform compatibility (Windows, macOS, Linux)
- Maintain lightweight, efficient performance
- Support direct peer discovery and connection management
