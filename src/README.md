# Source Code Directory

This directory contains the main source code for the x20chat P2P texting application.

## Purpose

The `src/` directory will house all Python modules and packages that implement:
- Peer-to-peer networking logic
- Message handling and routing
- User interface components
- Security and encryption features
- Core application functionality

## Structure

As development progresses, this directory will be organized into logical modules and subpackages to maintain clean separation of concerns and facilitate scalability.
